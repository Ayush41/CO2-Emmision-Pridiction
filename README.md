# 🌍 CO₂ Emissions Analysis & Forecasting (BRICS vs G7)

A research-driven project focused on analyzing historical CO₂ emissions and forecasting future trends (next 10 years) by comparing BRICS and G7 nations using key economic and environmental indicators.

---

## 📌 Project Overview

This project aims to:

- Analyze historical CO₂ emissions across countries
- Compare **BRICS vs G7 (developed nations)**
- Evaluate emissions using:
  - Total CO₂ emissions
  - CO₂ emissions per capita
  - GDP-based emissions
- Build predictive models to forecast emissions for the next 10 years
- Generate insights to support sustainability and policy discussions

## 📊 Key Metrics

The analysis is based on the following metrics:

- **Total CO₂ Emissions (kt / Mt)**
- **CO₂ Emissions per Capita**
- **GDP (current USD)**
- **CO₂ per GDP (carbon intensity)**
- **Year-wise growth rates**
- **Emission share by country/group**
